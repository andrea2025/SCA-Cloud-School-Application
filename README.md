# Documentation

1.Created a SCA-Cloud-School-Application repository
2.Cloned the repository into my computer using cli
3.Created and edited the index.html 
4.Created and edited the Dockerfile 
5.docker was installed 
6.build docker by running "docker build -t sca-cloud-school ." to create the docker images
7.run the command "docker images" to check if the the image is created
8.run the port using "docker run -p 5000:80 -d sca-cloud-school" which will map ngnix port to the system localhost port
9.To make changes to the dockerfile,stop the running docker container using "docker stop 57f00675f76c" specify the container ID,bulid docker and run the port again
