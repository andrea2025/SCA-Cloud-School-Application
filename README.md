# Documentation

Created a SCA-Cloud-School-Application repository
Cloned the repository into my computer using cli
Created and edited the index.html 
Created and edited the Dockerfile 
docker was installed 
build docker by running "docker build -t sca-cloud-school ." to create the docker images
run the command "docker images" to check if the the image is created
run the port using "docker run -p 5000:80 -d sca-cloud-school" which will map ngnix port to the system localhost port
To make changes to the dockerfile,stop the running docker container using "docker stop 57f00675f76c" specify the container ID,bulid docker and run the port again
